# Project Overview
**OctaNotes** is a music rhythm game developed in Unity.
The game features two lanes (top and bottom) with various note types: Tap, Long, and Chain notes.
There are also guide lines that do not affect judgement.

# Key Features
- **Lanes:** Top and Bottom.
- **Note Types:** Tap, Long, Chain.
- **Platform:** Unity (Darwin/macOS environment).
- **Language:** C#.
