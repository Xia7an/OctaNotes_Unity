(現在選択されているカードの2つ前) : (490, 14)
(現在選択されているカードの1つ前) : (433, 222)

(現在選択されているカードの2つ次) : (146, 890)

- 管理対象のカードは5つ以上になる場合がある。その場合、
(現在選択されているカードの3つ次)以降のカードはすべて(92, 1120)に表示
(現在選択されているカードの3つ前)以降のカードはすべて(547,-194)に表示
